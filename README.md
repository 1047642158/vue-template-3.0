# :monkey:vue-template:trollface::hatching_chick:

## :watermelon:项目设置 Project setup
```
npm install     安装运行依赖
```

### :whale:编译和热更新进行开发 Compiles and hot-reloads for development
```
npm run serve   运行项目并打开浏览器
npm run serves  以https运行项目并打开浏览器
```

### :whale2:编译并压缩用于正式环境 Compiles and minifies for production
```
npm run local   用于本地环境打包
npm run test    用于测试环境打包
npm run build   用于正式环境打包
```

### :bug:代码检查和修复文件 Lints and fixes files
```
npm run lint
